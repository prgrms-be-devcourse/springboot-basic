# week 1

### jar파일 생성

```shell
mvn clean package spring-boot:repackage 
```
### jar파일 실행

```shell
java -jar target/spring_week1-0.0.1-SNAPSHOT.jar --spring.profiles.active=dev 
java -jar target/spring_week1-0.0.1-SNAPSHOT.jar --spring.profiles.active=local
```
